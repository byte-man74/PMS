# Patient Management System (PMS)

PMS is a Django-based Patient Management System designed for demonstration purposes. This project is not intended for real-world deployment, and it may expose sensitive information, such as secret keys and environment variables, in the code for educational and showcase purposes.

## Features

- **Patient Management:** Manage patient information, including personal details, medical history, and appointments.
- **Appointment Scheduling:** Schedule and view appointments for patients.
- **User Authentication:** Secure user authentication system for accessing the system.
